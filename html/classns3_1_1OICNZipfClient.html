Detailed Description

OICN Zipf Client application, it generates request for the contents following Zipf-Mandelbrot Distribution. Here is the explanation of Zipf-Mandelbrot Distribution: http://en.wikipedia.org/wiki/Zipf%E2%80%93Mandelbrot_law. Client application send name resolution to ICN Manager. ICN Manager do the name resolution and direct source (ICN router or server) to send the content to the OICN Zipf Client.
